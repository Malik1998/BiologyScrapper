Bulk add subjects

Paste one name per line (or comma-separated), e.g. "Tom Hanks". Each name is looked up online (birth year, parents), then candidate photos are downloaded and filtered for all 4 slots (self 50-60, self 30-40, mother 50-60, father 50-60) — ready for manual labeling in the library afterwards. Names are processed one at a time; a failure on one name doesn't stop the rest.
